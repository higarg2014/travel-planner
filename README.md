# AI Trip Planner

An AI-powered travel planning application built with React, TypeScript, Material-UI, and Claude AI API.

## Features

- 🤖 **AI-Powered Generation**: Complete multi-day itineraries generated by Claude AI
- ✈️ **Flight Booking**: Outbound and return flight recommendations with prices
- 🏨 **Hotel Selection**: Accommodation recommendations with amenities
- 🎯 **Daily Activities**: Morning, afternoon, and evening activities for each day
- 🍽️ **Restaurant Recommendations**: Breakfast, lunch, and dinner suggestions
- 💰 **Budget Tracking**: Real-time budget monitoring with color-coded status
- 📱 **Responsive Design**: Works on mobile, tablet, and desktop

## Tech Stack

- **Frontend**: React 19 + TypeScript 6 + Vite 8
- **UI**: Material-UI v9
- **AI**: Claude API (Anthropic SDK)
- **Date Handling**: dayjs + Material-UI Date Pickers

## Quick Start

### 1. Install Dependencies

```bash
npm install
```

### 2. Set Up API Key

Create a `.env` file:

```bash
cp .env.example .env
```

Add your Anthropic API key:

```
VITE_ANTHROPIC_API_KEY=your_api_key_here
```

Get an API key at: https://console.anthropic.com/

### 3. Run Development Server

```bash
npm run dev
```

Open http://localhost:5173

### 4. Build for Production

```bash
npm run build
```

## Usage

1. **Enter Trip Details**:
   - Destination (e.g., "Paris, France")
   - Check-in and check-out dates
   - Number of travelers
   - Total budget
   - Optional preferences (activities, hotel type, food)

2. **Generate Itinerary**:
   - Click "Generate Itinerary"
   - AI creates a complete trip plan in seconds

3. **Review Your Trip**:
   - View flights with times and prices
   - See hotel details and amenities
   - Explore day-by-day activities and restaurants
   - Check budget status

4. **Plan Another Trip**:
   - Click "Plan Another Trip" to start over
